# 무한 스크롤로 구현한 Unsplash 사진첩
무한 스크롤을 구현해보기 위해 수행한 간단한 프로젝트입니다.

- 무한 스크롤 구현
  - `Intersection Observer`
- 사진 Api
  - Unsplash Api

## 실행화면
![ezgif com-gif-maker](https://user-images.githubusercontent.com/55550034/174434819-e28c96b4-dbb4-4b0c-9696-ace569d882b6.gif)

## 설치
`yarn install`

## 실행
`yarn start`
